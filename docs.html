Modding Helper Documentation

Clap / Whistle

Circle（type=1/5）のみを対象に、Whistle / Clap / Whistle + Clap の個数をカウントします。

チェックボックスを有効にすると、それぞれのタイムポイントも表示されます。

1msズレ

HitObject が snap 位置から ±1〜3 ms ずれている箇所を検出します。

このツールでは、.osu ファイルの beatLength をもとに snap 位置を計算し、小数点以下を切り捨てて比較しています。

±1 ms のズレは Beatmap Verifier や AiMod では表示されません。
ただし、±1 ms は rankable である点に注意してください。

Kiai比較

diff間の kiai time の総時間を比較します。

総時間が異なる場合、どの区間で kiai 状態が一致していないかを表示します。
また、kiai time の終了地点が明示されていない場合も警告します。

Beatmap Verifier では総時間の比較はできますが、具体的にどの区間が異なるかは確認できません。

Kiai Snap

kiai の ON/OFF が snap に乗っているかを確認します。

unsnapped な kiai line に加えて、1/8 snap などのズレも検出できます。

AiMod は unsnapped な timing point を検出できますが、1/8 snap などのズレは検出できません。

二重SV

指定した間隔以内にある、近すぎる SV line を検出します。

AiMod は同時刻の重複 SV は検出できますが、1 ms ズレの二重 SV は検出できません。

SV Volume

volume変更を伴う SV line が、HitObject に対して十分前倒しされているかを確認します。

HitObject と同時刻に volume変更を置いた場合、プレイヤーが 1 ms 早く叩くと新しい volume が適用されない場合があります。

この検出は AiMod や Beatmap Verifier では行えません。

赤線&緑線の一致

同時刻の赤線（BPM line）と緑線（SV line）を比較します。

  • volume の不一致
  • kiai ON/OFF の不一致
  • sampleset の不一致

この検出は AiMod や Beatmap Verifier では行えません。

SampleSet

TimingPoints

赤線・緑線では、sampleset 2（Soft）または 3（Drum）が含まれる場合に表示します。

sampleset 1（Normal）は検出されません。

HitObjects

ノーツ、スライダー、スピナーでは、sampleset 2（Soft）または 3（Drum）が含まれる場合に表示します。

sampleset 0（Auto）および 1（Normal）は検出されません。
