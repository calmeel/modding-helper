# Modding Helper
![alt text](images/image.png)
Web-based modding helper for osu!taiko.

## 🌐 Web App
👉 https://calmeel.github.io/modding-helper/

## 🛠 Features
- 1ms offset detection
- Kiai mismatch detection
- SV / volume checks
- Red & Green line consistency check
- Sampleset validation
- Supports `.osu` and `.osz`
- Multi-difficulty analysis
