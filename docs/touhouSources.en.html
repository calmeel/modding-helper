The Source check compares the beatmap Source field against this list to detect:

  • Spelling mistakes
  • Incorrect full-width / half-width spaces
  • Missing periods
  • Incorrect wave dash characters

Notes
This list is used only for Source formatting assistance.
It does not guarantee that every Touhou-related work or derivative title is supported.

Some titles intentionally use:
  • Full-width spaces (　)
  • Full-width wave dashes (～ / 〜)
  • Specific capitalization
  • Trailing periods
Therefore, visually similar Source values may still be treated as mismatches.
