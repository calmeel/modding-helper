読み取れる傾向

  • Kantan → Futsuu は中央値が x1.83 と高く、 低難易度では Ratio が大きくなりやすい傾向があります。
  • Futsuu → Muzukashii は中央値 x1.40、 Muzukashii → Oni は中央値 x1.29 で、 difficulty が上がるほど Ratio の中心値が下がっています。
  • Oni → Inner Oni+ は中央値 x1.24、 Inner Oni+ → Inner Oni+ は中央値 x1.12 で、 高難易度側では小さい Ratio でも自然な spread になりやすいです。
  • ゲスト難易度を含む比較では、特に Kantan → Futsuu の標準偏差が大きく、 mapper ごとの設計差が Ratio に出やすい可能性があります。
  • Below Kantan → Kantan は件数が少ないため、 この統計だけで基準値を決めるには不十分です。

閾値設定について

Modding Helper の Warning / Error 閾値は、この統計結果を参考に設定しています。

  • Warning：各 difficulty pair において、サンプル内でおおよそ下位 5% / 上位 5% に入る Ratio を目安に設定
  • Error：各 difficulty pair において、サンプル内でおおよそ下位 1% / 上位 1% に入る Ratio を目安に設定
  • Ratio が 1.00 未満：上位 difficulty の Circle 数が前 difficulty より少ないため、原則として Error 扱い

ただし、実際の閾値は統計値をそのまま機械的に採用したものではなく、 見やすさや運用上の分かりやすさを考慮して、小数第2位付近に丸めています。

Warning / Error は「不正」や「unrankable」を示すものではなく、 人間が確認すべき可能性のある箇所を示す補助表示として扱ってください。

注意事項
この統計は、Ranking Criteria 上の公式基準ではありません。
Warning / Error の閾値は、実際の ranked 譜面の傾向を参考にした、 Modding Helper 独自の確認用目安です。

表示された数値は、ノーツ数 Ratio の傾向を確認するための補助情報であり、 実際の difficulty spread を完全に表すものではありません。
